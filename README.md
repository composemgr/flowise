## 👋 Welcome to flowise 🚀

Flowise - Build LLM apps visually

## 📋 Description

Build LLM apps visually

## 🚀 Services

- **app**: Flowise (`flowiseai/flowise:latest`)

## 📦 Installation

```shell
composemgr install flowise
```

## 🔧 Configuration

```shell
TZ=America/New_York
BASE_HOST_NAME=flowise.example.com
```

## 🌐 Access

- **Flowise**: http://localhost:3000

## 📂 Volumes

- `./rootfs/data/flowise` - Application data

## 🔐 Security

- Change default passwords
- Configure HTTPS with reverse proxy
- Regular backups
